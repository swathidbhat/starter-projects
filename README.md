# starter-projects

This repo holds my beginner Python projects. These take more code than a single
math problem and tend to combine a few ideas, like game logic or pulling data
from a service.

## Projects

| Project | What it does | Original date |
| --- | --- | --- |
| `tic-tac-toe/Tic-Tac-Toe.ipynb` | Takes each player's board position, updates the board, and prints the winner or a tie | 2020-10-21 |
| `Google Maps Low Reviews - Trip Planning - Py 3.ipynb` | Given an app ID and trip-planning keywords, fetches matching 1-star Google Maps reviews and saves them to a CSV | 2024-04-22 |
