# starter-projects

This repo holds my beginner Python projects. These take more code than a single
math problem and tend to combine a few ideas, like game logic or pulling data
from a service. Each project lives in its own folder with its own README.

## Projects

### Tic-Tac-Toe

- A two-player game of Tic-Tac-Toe. Players take turns entering a board
  position, the board redraws after each move, and the program announces a win
  or a tie.
- See [`tic-tac-toe/`](tic-tac-toe/) for setup and gameplay.
- Original date: 2020-10-21. Folder: `tic-tac-toe/`.

### Google Maps Low Reviews

- A study of user pain points with Google Maps during trip planning. The
  notebook extracts 1-star reviews that mention trip-planning keywords, so the
  complaints are easy to read in one place.
- See [`google-maps-low-reviews/`](google-maps-low-reviews/) for setup and
  details.
- Original date: 2024-04-22. Folder: `google-maps-low-reviews/`.
